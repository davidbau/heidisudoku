Wie man spielt

Blättern Sie durch Rätsel mit den Schaltflächen Weiter und Zurück oder den Tasten N und P. Sie können an den Rätseln in beliebiger Reihenfolge arbeiten und später zu jedem Rätsel zurückkehren.

Um ein Rätsel auszufüllen, klicken Sie auf eine Zahl rechts und dann auf ein beliebiges Feld, um diese Zahl mit Bleistift einzutragen. Die Regeln des Sudoku sind, dass jede Ziffer 1-9 einmal in jeder Zeile, Spalte und jedem 3x3-Block erscheinen muss.

Tastaturkürzel

Sie können auch die Tastatur verwenden. Zeigen Sie auf ein Feld oder verwenden Sie die Pfeiltasten ← ↑ ↓ →, um das gepunktete blaue Rechteck zu bewegen, und tippen Sie dann eine Ziffer 1 - 9, um es auszufüllen. Das Eingeben mehrerer Zahlen in einem Feld markiert alle diese Zahlen, aber eine doppelt angetippte Zahl löscht alle Markierungen und schreibt die Zahl als Antwort. Verwenden Sie die Leertaste oder die 0-Taste, um das ausgewählte Feld erneut zu löschen.

Die Kommataste , öffnet ein Menü, mit dem Sie mehrere Zahlen markieren können. Bei angezeigtem Menü ermöglicht die H-Taste, markierte Zahlen gelb hervorzuheben. Die M-Taste markiert automatisch Zahlen, die nicht in direktem Konflikt in der Zeile, Spalte oder dem Block stehen, und die Schaltfläche Markierungen füllt diese für jedes Feld aus.

Hinweise erhalten

Klicken Sie auf eine Farbe links, um Felder auf diesem Schwierigkeitsniveau anzuzeigen: violette Felder sind am einfachsten, und rote Felder sind am schwierigsten. Klicken Sie auf die Schaltfläche Farbe, um alle Farben anzuzeigen.

Halten Sie die Schaltfläche Hinweis gedrückt, um einen bestimmten Hinweis zu erhalten. Rote Felder weisen auf Fehler hin. Blaue Felder sind Orte, die das Rätsel auf irgendeine Weise einschränken, und grüne Felder sind Orte, an denen eine Zahl ausgefüllt werden kann oder wo die Möglichkeiten eingegrenzt werden können.

Wenn Sie immer noch feststecken, können Sie einen expliziten Hinweis versuchen, indem Sie ctrl gedrückt halten, während Sie auf die Schaltfläche Hinweis klicken. Manchmal sind die gezeigten Strategien sehr subtil: Lernen Sie, nackte Paare und Überschneidungen zu erkennen, und sehen Sie, ob Sie die versteckten Paare oder Tripel finden oder einen X-Wing, Y-Wing oder Schwertfisch erkennen können.

Fehler überprüfen und korrigieren

Die Schaltfläche Überprüfen kann verwendet werden, um schnell zu überprüfen, ob Sie Fehler oder widersprüchliche Felder eingetragen haben, ohne zu verraten, was die Fehler sind.

Das Halten von ctrl beim Klicken auf Überprüfen zeigt Ihnen die Position eines Fehlers.

Wenn Sie einen Fehler entdecken, können Sie die „Zurück"-Schaltfläche des Browsers verwenden, um so viele Züge wie nötig rückgängig zu machen.

Der Timer

Das Spiel verfolgt die Zeit, die Sie benötigt haben, um ein Rätsel zu lösen. Drücken Sie die Schaltfläche Timer, um die verstrichene Zeit und den Hinweiszähler anzuzeigen.

Wenn Sie sich von einem Rätsel entfernen und nicht möchten, dass die gesamte Leerlaufzeit gegen Sie gezählt wird, keine Sorge: Drücken Sie die „Aktualisieren"-Schaltfläche in Ihrem Browser, und der Timer kehrt zu der Zeit zurück, zu der Sie Ihren letzten Zug gemacht haben. Ebenso wird, wenn Sie ein Spiel speichern, die gespeicherte Zeit die Zeit des letzten gemachten Zugs sein.

Rätsel speichern und teilen

Rätsel können auch mit der Schaltfläche Datei gespeichert und geladen werden. Gespeicherte Spiele werden durch Bewertung, Datum und Fortschritt identifiziert, oder Sie können sie selbst umbenennen.

Wenn Sie ein Juwel eines Spiels entdecken, das Sie teilen möchten, erstellt der Link Als URL speichern eine kurze URL für Ihr aktuelles Rätsel, die per E-Mail versendet werden kann. Alle Markierungen und Farben, die Sie auf dem Spiel platzieren, werden zusammen mit der URL gespeichert, sodass Sie einen besonders cleveren Sudoku-Schritt mit anderen Rätselfreunden teilen können.

Eigene Rätsel eingeben

Wenn Sie mit der rechten Maustaste klicken oder ctrl gedrückt halten, während Sie auf ein Feld klicken, können Sie das Rätsel jederzeit ändern. Das Halten von ctrl beim Klicken auf Löschen löscht das gesamte Rätsel, und Sie können ein neues Rätsel mit der Tastatur oder Maus eingeben. Beachten Sie, dass einige Zahlen nicht eingegeben werden können, da der Hinweisblock nicht zulässt, dass Sie ein Rätsel ohne Lösung eingeben.

Sobald Sie ein Rätsel mit einer eindeutigen Lösung eingegeben haben, wird eine Schwierigkeitsbewertung angezeigt und die Tastatur wechselt in den Spielmodus.

Mathematisch minimale Rätsel

Normalerweise haben die Rätsel in Heidis unendlichem Sudoku-Block 25-30 Hinweise. Natürlich genießen Sie ein etwas längeres Spiel, wenn es weniger Hinweise gibt.

Die kleinste Anzahl von Hinweisen, die bei einem lösbaren Sudoku-Spiel gegeben werden kann, wird auf 17 geschätzt, und Gordon Royle pflegt eine Datenbank aller bisher entdeckten 17-Hinweis-Rätsel. Wenn Sie ctrl gedrückt halten, während Sie auf Datei klicken, wird eines dieser 17-Hinweis-Rätsel aus dem Internet geladen.

Schwierigkeitsbewertungen

Sudoku-Rätsel werden auf einer 24-Stufen-Skala von „Anfänger" (am einfachsten) bis „Verwirrend" (am schwierigsten) bewertet. Die vollständige Skala:

1: Anfänger
2: Leicht
3: Einfach
4: Grundlegend
5: Mäßig
6: Knifflig
 7: Clever
8: Rätselhaft
9: Subtil
10: Schwierig
11: Verwickelt
12: Dornig
 13: Verblüffend
14: Kompliziert
15: Verwirrend
16: Unklar
17: Labyrinthisch
18: Hartnäckig
 19: Abstrus
20: Ärgerlich
21: Rätselhaft
22: Formidabel
23: Teuflisch
24: Verwirrend

Rätsel, die gelöst werden können, indem man einfach direkte Konflikte vermeidet, sind „Grundlegend" oder einfacher, abhängig davon, wie leicht die Einschränkungen zu erkennen sind. „Subtile" Rätsel erfordern ein oder zwei clevere Ableitungen. Und es wird von dort aus schwieriger: „Verwirrende" Rätsel beinhalten das Entwirren einer Masse komplexer Hinweise.

Danksagungen

Heidis Hinweisblock wurde in HTML und Javascript von Heidis Ehemann David Bau geschrieben, unter Verwendung von jQuery von John Resig mit Ben Almans praktischem BBQ-Plugin (als Fork von Edison Wong). Joe Princes wunderbare Handlee-Schrift liefert handgeschriebene Ziffern und Steve Mattesons Open Sans wird für die Rätselnummern verwendet. Jojo Mendoza hat das gelbe Textmarkersymbol bereitgestellt; und Gordon Royles mathematisch schöne minimale Sudoku-Sammlung liefert minimale 17-Hinweis-Rätsel, wenn Sie ctrl gedrückt halten, während Sie auf Datei klicken.

Die Quellen für Heidis Sudoku-Hinweisblock sind derzeit nicht zur Wiederverwendung lizenziert, aber sie können hier auf Github gelesen werden.
